## 🎭 Animation Sequence

### Page Load
```
1. Hero fades in from bottom (0.8s)
   ├─ Avatar scales in (1s, delay 0.2s)
   ├─ Name fades in (1s, delay 0.4s)
   ├─ Title fades in (1s, delay 0.6s)
   └─ Description fades in (1s, delay 0.8s)

2. Sections fade in on scroll
   └─ Each section: fade from bottom (0.6s)

3. Timeline items
   └─ Staggered fade from left (0.6s each, 0.1s delay)

4. Background gradient
   └─ Continuous 15s cycle animation
```

### Interactions
```
Hover on Card:
├─ Transform: translateY(-6px)
├─ Shadow: Increases
└─ Duration: 0.3s

Hover on Button:
├─ Transform: translateY(-2px)
├─ Background: Lighter gradient
└─ Duration: 0.3s

Hover on Skill Badge:
├─ Transform: translateY(-3px)
├─ Stars: Scale 1.2x
└─ Duration: 0.3s

Click on Button:
├─ Ripple effect from click point
└─ Expands to 300px diameter
```

## 🎨 Color Usage Map

```
Hero Background:
├─ Gradient: #667eea → #764ba2 → #f093fb
└─ Animation: 15s color shift

Section Backgrounds:
├─ Odd sections: White (#ffffff)
└─ Even sections: Light gradient (rgba(102, 126, 234, 0.03))

Card Colors:
├─ Background: rgba(255, 255, 255, 0.92)
├─ Border: rgba(255, 255, 255, 0.3)
└─ Shadow: rgba(0, 0, 0, 0.08)

Text Colors:
├─ Headings: #1f2937 or gradient
├─ Body: #4b5563
├─ Light: #6b7280
└─ Gradient Text: #667eea → #764ba2

Accent Colors:
├─ Primary Buttons: #667eea → #764ba2
├─ Secondary Buttons: #f093fb → #f5576c
├─ Timeline Dots: #667eea
└─ Success: #10b981
```
